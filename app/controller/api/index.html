使い方

GoogleスプレッドシートのIMPORTDATA関数の引数に以下のURLパスを指定してください。


URLパス 例 取得できるデータ
/ / 使い方説明
/howto /howto 使い方説明（WEBブラウザ）Chromeなどをつかってアクセスしてください
/financial /financial 全銘柄基本情報
/statement?code={{銘柄コード}} /statement?code=7203 全年分財務情報（銘柄コード指定）
- {{銘柄コード}}は取得したい銘柄を4桁または5桁で指定
/price?code={{銘柄コード}} /price?code=7203 全年分株価情報（銘柄コード指定）
- {{銘柄コード}}は取得したい銘柄を4桁または5桁で指定
/price?ymd={{日付}} /price?ymd=2024-09-02 全銘柄株価情報（日付指定）
- {{日付}}はYYYY-MM-DDの形式で指定
/price?code={{銘柄コード}}&ymd={{日付}} /price?code=7203&ymd=2024-09-02 株価情報（銘柄コード・日付指定）
- {{銘柄コード}}は取得したい銘柄を4桁または5桁で指定
- {{日付}}はYYYY-MM-DDの形式で指定
